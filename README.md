本对外接口文档由 Carl 维护，发现任何错误都可以反馈。

如需本地运行，请进行下面的步骤。

下载本项目到本地：

    git clone https://github.com/CoolKit-Technologies/eWeLink-API.git

安装「docsify」脚手架：

    npm i docsify-cli -g

在当前目录（/eWeLink-API 上层目录），输入命令：

    docsify serve eWeLink-API

打开浏览器访问：http://localhost:3000 或 http://127.0.0.1:3000
